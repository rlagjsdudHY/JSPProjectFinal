# JSPProjectFinal
JSP Model1 팀프로젝트 커뮤니티게시판

JSP,Tomcat 9.0,Eclipse IDE (ver.2021-12 (4.22.0))

작업기간 2023.04.14 ~ 2023.04.25

========== 직접 구현한 내용 ===========

회원 관리 - 비밀번호 변경 , 회원탈퇴, 
게시판 - 갤러리 게시판, 첨부파일 수정
어드민페이지 - 어드민 로그인 , 게시판 관리(수정,삭제,조회가능) , 회원 강제탈퇴,회원정보 관리
